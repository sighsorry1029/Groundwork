# Changelog

## 1.0.1

- Added beehive hover details for honey capacity, Farming capacity bonus, cover, pollination, night/rain honey rates, and next honey timing.
- Added Farming-scaled beehive capacity and Farming skill gain from honey harvests.
- Added beehive pollination bonuses for plant growth, foraging respawn, and honey production, with rain disabling pollination while loaded.
- Added rain growth/respawn bonuses and compact plant/foraging hover multiplier summaries.
- Made grid planting always available independently from mass planting, while mass planting still scales by Farming level.
- Added terrain tool range scaling, grid precision previews, scalable pickaxe terrain digging, and tool HUD controls.
- Added ZenBeehive container compatibility so honey removed from beehive containers counts as Groundwork harvest.
- Added English/Korean localization tokens and cleaned README/config documentation.
- Reduced structure risk by consolidating plant growth patches, moving terrain tool config ownership, and removing always-on compatibility aliases.
